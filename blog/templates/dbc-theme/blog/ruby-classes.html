odometer method can access advence km variable, that is why we can run our previous code, we need a instance variable so... 


  class Awsome_car
    def initialize
      @km = 0
      puts "I'm a new car, i have #{@km} km"

    end

    def advance
      @km = @km + 1
      km = 1
      puts "Move #{km}km"
    end

    def steer_right
      puts "Turn righ 45 degrees"
    end

    def steer_left
      puts "Turn left 45 degrees"
    end

    def odometer
      puts "I have #{@km} in total"
    end

  end

Notice @km and km are to different variables, and two different kinds one, km is just a method variable, and the @km is an instance method,this is a variable available in a "instance" (copy) of the Awsome_card class
